# Changelog

## [Initial Release] - 2026-05-04

- **Exchange Rates**: Browse popular crypto exchange directions with best rates
- **Quick Search**: Type currency names to instantly find exchange rates (e.g. "usdt trc20 monobank")
- **Check Rate**: Form with FROM/TO currency dropdowns, auto-calculated amounts, country/city for cash directions
- **Direction Rates**: Compare all exchangers for a currency pair with sorting (best rate, trust rating, favorites)
- **Exchangers**: Browse catalog of 100+ exchangers with trust scores and ratings
- **My Favorites**: View saved favorite directions and exchangers
- 87 bundled currency icons for instant display
- Trust status badges (Diamond, Premium, Gold, New) with icons and explanations
- Exchanger parameter icons (Manual, Verify, Registration, Card Verify, Floating)
- Favorites & Blacklist persisted between sessions
- Exchange redirect via Kurslog tracking API
- Multilingual: Ukrainian, Russian, English (auto-detected from system language)
- Dedicated cached API (`/api/raycast`) with rate limiting for optimal performance
