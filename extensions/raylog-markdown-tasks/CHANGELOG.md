# Changelog

## [Initial Release] - 2026-05-11

- Manage focused local tasks from Raycast with a single markdown-backed note.
- Create, edit, log work, and move tasks through `To Do`, `In Progress`, `Done`,
  and `Archived`.
- Review tasks through saved list layouts, filtering, search, and menu bar access
  for current work.
- Recover from missing, malformed, or outdated managed storage blocks from inside
  the extension.
