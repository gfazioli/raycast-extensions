## [Initial Release] - 2026-05-11

### Added

- Initial release of Tibia Helper extension
- **Search Item NPC Price**: Find the highest NPC sell price for Tibia items
  - Search functionality for Tibia items
  - Display highest NPC sell price for each item
  - View all buyers sorted by price
  - Copy price information to clipboard
  - Stackable item indicators
  - Comprehensive database of Tibia items and NPC buyers
- **Rashid Location**: Show which city Rashid is currently in
  - Current location with countdown timer
  - Next location preview
  - Full weekly schedule
  - Timezone support (CET/CEST)
- **Search Imbuements**: View imbuement tiers and required materials
  - 15 different imbuements covering all categories
  - All three tiers (Basic, Intricate, Powerful)
  - Complete material lists with quantities
  - Equipment type information
  - Search by imbuement name or variant name
